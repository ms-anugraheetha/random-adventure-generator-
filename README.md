# 🌱 Cozy Micro-Adventures – Interactive React App:

A calm, micro-interaction–driven web experience built with React + Vite.
The app presents users with gentle, low-effort prompts designed to encourage
grounding, comfort, and intentional pauses in everyday life.

This project emphasizes frontend UX, microcopy design, and state-driven UI
over complex business logic or heavy frameworks.

## Features:

 Randomized micro-adventure prompts

 Card-based prompt presentation

 State-driven UI updates

 Minimal, calm visual design

 Soft transitions and animations

 Fully responsive (mobile-first)
 

## Tech Stack:

React

Vite

JavaScript (ES6+)

CSS (custom styling & transitions)


## What I Learned:

Designing UI around micro-interactions

Structuring reusable React components

Managing state for dynamic content rendering

Writing effective microcopy for user interfaces

Building calm, non-intrusive user experiences
