# 🌱 Cozy Micro-Adventures – Interactive React App

A calm, micro-interaction–driven web experience built with **React + Vite**. 
The app presents users with gentle, low-effort prompts designed to encourage grounding, comfort, and intentional pauses in everyday life.

This project emphasizes **frontend UX**, **microcopy design**, and **state-driven UI** over complex business logic.

---

## ✨ Features

- 🎲 **Randomized micro-adventure prompts** – Generate gentle, personalized suggestions
- 🎴 **Card-based prompt presentation** – Clean, intuitive UI
- ⚡ **State-driven UI updates** – Smooth, responsive interactions
- 🎨 **Minimal, calm visual design** – Focused on user comfort
- 🌀 **Soft transitions & animations** – Custom CSS + Framer Motion
- 📱 **Fully responsive** – Mobile-first approach

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation

```bash
git clone https://github.com/ms-anugraheetha/random-adventure-generator-.git
cd random-adventure-generator-/random-adventure-generator
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will open at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

## 💡 Key Concepts

### State Management
Uses React hooks (`useState`, `useEffect`) to manage prompt state and UI transitions.

### Animations
Custom CSS transitions and Framer Motion for smooth, performant animations without animation libraries.

### Responsive Design
Mobile-first CSS with media queries for seamless experience across all devices.
